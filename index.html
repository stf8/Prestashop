    Addons for Prestashop

    This fork supports small addons to Prestashop :

    • new hooks in the backoffice import controller;
    • background bulk CSV import.

    How to apply

    If you don't already have your very own Prestashop running then you can get a patched Prestashop by clicking the links at top of the page.

    Alternatively you can apply a single patch to your 1.5.2 tree (only this release is provided).

    prestashop$ patch -p1 < /x/y/z/where/is/the/patch.file

    Special notes

    I didn't use override since most of modification occurs in the middle of pretty critical methods this would have lead to duplicate most of those methods.

    There is one recurrent problem in using overrides : array defininition in methods (e.g. render_form). Most of time you will see :

    $this->an_array = array( 'x' => 'y');

    While one should expect a way to merge a possible predefined array (typically from within the override) :

    public function render_form() {
  $this->an_array = array( 'foo' => 'bar' );
  parent::render_form();
}

    Whatever, the patch is pretty easy to apply.
